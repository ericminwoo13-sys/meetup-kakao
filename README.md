# 📍 중간 지점 찾기 (카카오맵 버전)

---

## 🔑 STEP 1 — 카카오 API 키 발급 (5분)

### 1-1. 카카오 개발자 콘솔 접속
https://developers.kakao.com → 로그인 (카카오 계정)

### 1-2. 애플리케이션 추가
1. **내 애플리케이션 → 애플리케이션 추가하기**
2. 앱 이름: `meetup-midpoint` (아무 이름)
3. 회사/팀 이름 입력 후 저장

### 1-3. 플랫폼 등록
1. 앱 클릭 → 좌측 **플랫폼**
2. **Web 플랫폼 등록**
3. 사이트 도메인: `http://localhost:3000` 입력 → 저장

### 1-4. 키 확인
앱 → **앱 키** 메뉴에서:
- **JavaScript 키** → 지도 표시용
- **REST API 키** → 서버 장소 검색용

두 키 모두 복사해 두세요.

### 1-5. Kakao Local API 활성화 (중요!)
1. 앱 → 좌측 **카카오 API → 활성화 설정**
2. **카카오맵 → 사용** 켜기

---

## ⚙️ STEP 2 — 환경 설정

### .env 파일 생성
프로젝트 루트에 `.env` 파일 생성:

```
KAKAO_JS_KEY=발급받은_JavaScript_키
KAKAO_REST_KEY=발급받은_REST_API_키
PORT=3000
```

---

## 🚀 STEP 3 — 실행

```bash
# 패키지 설치 (최초 1회)
npm install

# 서버 시작
npm start
```

브라우저에서 → http://localhost:3000

---

## 💡 사용 방법

1. 친구 이름 옆 검색창에 위치 입력 (예: `서울역`, `강남구청`, `춘천 명동`)
2. 드롭다운에서 선택 → 지도에 마커 표시
3. `+ 친구 추가` 버튼으로 3명, 4명... 추가 가능
4. **중간 지점 찾기** 버튼 클릭
5. **거리 기반** / **이동 시간 기반** 두 가지 결과 비교
6. 결과 카드에서 주변 지하철역 배지 확인
7. 맛집 / 카페 / 지하철 / 볼거리 / 문화 탭으로 주변 장소 5개 확인
